FactMagnifier

An interactive factual application designed for people who thrive on knowledge! The application is fully functional and available on the App Store, created from scratch using various technologies. With this application, users can scroll through an endless amount of interesting facts and choose from the various available categories. Try it for yourself!

  • Expo React-Native:

    The main framework used to create the app includes the development of reusable components, routing, design, and more!

  • Push Notifications:

    The application uses the Expo Notifications system to fetch the user's push token for receiving daily notifications about interesting facts!

  • Backend-Server:

    The application interacts with a backend server, which was created from scratch using. The backend handles GET requests from the front-end to fetch a 'fact' object from an external API and POST requests to store the push token in the database.

  • Firebase:

    The backend interacts with Firebase Firestore and Firebase Functions to store push tokens and interact with an external API for fetching data. The Firebase functions are responsible for various tasks, including interacting with the front-end, fetching data from the facts database, sending push notifications, deleting unused tokens, adding new ones, and more!
